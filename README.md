📘 OPERATING-SYSTEM-ALGOS

A collection of Operating System algorithms implemented in C/C++ for academic learning and practical understanding. This repository covers core OS concepts like process scheduling, synchronization, memory management, and disk scheduling.

🚀 Overview

Operating systems manage hardware and software resources and provide a platform for applications to run efficiently Operating Systems.

This repository focuses on implementing fundamental OS algorithms to understand:

How processes are scheduled
How memory is managed
How deadlocks occur and are handled
How disk requests are optimized
📂 Algorithms Included
🧠 CPU Scheduling
First Come First Serve (FCFS)
Shortest Job First (SJF)
Priority Scheduling
Round Robin
💽 Disk Scheduling
SSTF (Shortest Seek Time First)
FCFS Disk Scheduling
SCAN / LOOK (if added)
📄 Page Replacement
FIFO (First In First Out)
LRU (Least Recently Used)
Optimal Algorithm
🍽️ Synchronization Problems
Dining Philosophers Problem
Producer-Consumer Problem
🛠️ Technologies Used
C / C++
Basic Data Structures
